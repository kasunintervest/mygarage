Install RVM
----------------------------------
curl -sSL https://get.rvm.io | bash


Install Ruby
----------------------------------
rvm install ruby-2.3.6
rvm --default use ruby-2.3.6


Install MyGarage App
----------------------------------
git clone https://github.com/kasunintervest/mygarage.git


# RAILS COMMANDS
----------------------------------
bundle install
rails db:migrate
